# CodeAlpha_StudentGradeTracker
Description
Student Grade Tracker is a Java-based application designed to manage and monitor students' academic performance efficiently. It allows users to add, update, and calculate grades for multiple subjects, providing a clear overview of each student's performance. The project demonstrates object-oriented programming concepts and file handling in Java.

# Features
- Add, update, and delete student records
- Input grades for multiple subjects
- Automatically calculate grade classification
- Display performance reports for individual students
- Save and load student data using file handling

# Technologies Used
- Java (Core Java, OOP concepts)
- File handling for data storage
- Console-based user interface

# How to Run
1. Clone or download the repository.
2. Open the project in your preferred Java IDE (e.g., Eclipse, VS Code).
3. Compile and run the main Java file (`Main.java` or similar).
4. Follow the on-screen menu to manage student grades.
